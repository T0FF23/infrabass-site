# INFRABASS - Vinyls & Skateboards Shop

Site e-commerce pour la vente de vinyles et skateboards.

## Installation

```bash
npm install
```

## Développement local

```bash
npm run dev
```

Ouvre http://localhost:5173 dans ton navigateur.

## Déployer sur GitHub Pages

```bash
npm run deploy
```
